## Execute appliction

* `$ bundle install`
* `$ rake db:create`
* `$ rake db:migrate`
* `$ rails server`

## Execute test using RSpec
* `$ bundle install`
* `$ rspec --format Fuubar spec`

## Execute security check using Brakeman
* `$ brakeman -o report`
* `$ open report`
